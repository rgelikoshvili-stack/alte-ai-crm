## Local Demo

```powershell
cd C:\tmp\alte-ai-crm\backend
copy .env.local.example .env
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
alembic upgrade head
python -m app.scripts.setup_local_demo
uvicorn app.main:app --reload
```

Second terminal:

```powershell
cd C:\tmp\alte-ai-crm\widget
python -m http.server 5500
```

Open:

```text
http://127.0.0.1:5500/demo.html
```

Smoke test:


## Known Limitations

- Not production-ready.
- No production deployment yet.
- No real Alte website embed yet.
- No WhatsApp, Messenger, Instagram, or Email integration yet.
- No scraping or crawling.
- Claude real mode is available, but local demo uses mock by default.
- Static frontend/widget only.
- Security is local/MVP level, not final enterprise policy.
- Knowledge seed is manually curated demo/governance content and must be reviewed before public use.
