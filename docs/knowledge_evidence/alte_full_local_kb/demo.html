Alte AI CRM Chat Widget Demo

Run the backend on http://127.0.0.1:8000, then open this page through a local static server.

  1. Backend: uvicorn app.main:app --reload
  2. Widget server: python -m http.server 5500
  3. Diagnostics: http://127.0.0.1:8000/diagnostics/local-demo

API base URL: http://127.0.0.1:8000

Changing this selector stores a new demo config and reloads the page. Use join.alte.edu.ge to test the international admissions default.
