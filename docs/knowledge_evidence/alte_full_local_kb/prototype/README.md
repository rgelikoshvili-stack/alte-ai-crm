# Alte AI Assistant Prototype

Static prototype for an Alte University website chat widget.

## What it demonstrates

- Bottom-right website chat widget
- Georgian/English language switch
- Quick replies for programs, law, admission, funding, international students, and human handover
- Source-aware answers with official Alte URLs
- Lead preview when user enters phone or email
- CRM-oriented behavior: general info does not create a lead, admission/contact intent can create a lead

## Files

- `index.html` - demo page and widget markup
- `styles.css` - Alte-inspired visual style
- `app.js` - mock chatbot flow and lead-capture behavior

Open `index.html` in a browser to preview.
